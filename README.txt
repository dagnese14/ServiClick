 es una moderna e intuitiva aplicación de punto de venta (POS) diseñada para optimizar las operaciones de negocios de comida rápida. Desarrollada con Python y el framework Kivy, la aplicación ofrece una interfaz de usuario sencilla y táctil, ideal para su uso tanto en computadoras de escritorio como en dispositivos móviles.

La aplicación es una solución integral y robusta para la gestión de ventas, inventario y relaciones con los clientes.

Características Principales
Gestión Eficiente de Ventas: Procesa pedidos de manera rápida y maneja distintos métodos de pago (USD, Bs.).

Control de Inventario en Tiempo Real: Descuenta automáticamente el stock de productos con cada venta, ofreciendo un seguimiento preciso del inventario.

Gestión de Clientes: Permite asociar ventas a clientes específicos para un mejor seguimiento y servicio.

Tasa de Cambio Dinámica: Convierte precios entre dólares y bolívares en tiempo real.

Reportes Avanzados: Genera informes de ventas detallados con visualizaciones gráficas (barras, circulares y de línea) para ayudarte a analizar tendencias de venta, identificar los productos más vendidos y entender las preferencias de pago.

Arquitectura Escalable: Construida sobre una base de datos SQLite, la aplicación es fácil de extender y adaptar a las necesidades de tu negocio.
